SmartShopping 接口增强与 Skill 设计

C端购物助手 API 增强方案与完整 Skill 配置文档

v1.2.0

一、搜索接口增强

接口：POST /open/products/search

原始接口参数

参数类型说明示例
keywordString搜索关键词"蓝牙耳机"
pageInteger页码1
pageSizeInteger每页数量10

新增参数

参数类型必填说明示例
priceMinFloat否价格下限50
priceMaxFloat否价格上限500
sortByString否排序方式price_asc / price_desc
sortBy 可选值：
• price_asc - 价格升序（最便宜优先）
• price_desc - 价格降序（最贵优先）
• 默认不传时按综合排序

使用场景映射

用户表达参数组合
"最便宜的蓝牙耳机"{keyword:"蓝牙耳机", sortBy:"price_asc"}
"100元左右的蓝牙耳机"{keyword:"蓝牙耳机", priceMin:80, priceMax:120}
"不超过500元的蓝牙耳机"{keyword:"蓝牙耳机", priceMax:500}

二、商品详情接口增强

接口：GET /open/products/{productId}

字段命名严格遵循淘宝客原文档 taobao.tbk.item.info.get

新增字段（全部新增，不分优先级）

字段说明展示规则
num_iid商品ID内部
title商品标题（长标题）展示
pict_url商品主图展示
small_images商品小图列表可选
reserve_price商品一口价格可选
zk_final_price折扣价展示
user_type卖家类型(0集市/1商城/3特价版)展示
provcity商品所在地展示
item_url商品链接内部
seller_id卖家ID内部
volume30天销量展示 ⭐高价值
nick店铺名称展示
cat_name一级类目名称可选
cat_leaf_name叶子类目名称可选
is_prepay是否加入消费者保障可选
shop_dsr店铺DSR评分展示 ⭐高价值
ratesum卖家等级可选
i_rfd_rate退款率是否低于行业均值可选
h_good_rate好评率是否高于行业均值可选
h_pay_rate30成交转化是否高于行业均值可选
free_shipment是否包邮展示 ⭐高价值
material_lib_type商品库类型内部
superior_brand是否品牌精选展示
hot_flag是否热门商品可选
⚠️ 注意：以上字段名严格遵循淘宝客原文档命名，如 nick（店铺名称）而非 shop_name，volume（销量）而非 sales_30d。新增字段时请核对原文档确保一致性。

五、字段展示规则

✅ 必须展示

  • 商品标题（title）
  • 价格（zk_final_price，折扣价优先）
  • 商品主图（pict_url）
  • 店铺名称（nick）
  • 购买链接（推广链接）
  • 30天销量（volume）
  • 店铺DSR评分（shop_dsr）
  • 是否包邮（free_shipment）
  • 卖家类型（user_type，天猫标识）

⚠️ 可选展示

  • 优惠券信息
  • 商品所在地（provcity）
  • 小图列表（small_images）
  • 类目信息

❌ 不展示给用户

  • 各种ID（num_iid、seller_id等内部字段）
  • 佣金信息（不透出给C端用户）
  • 技术字段（material_lib_type等）
⚠️ 佣金信息不透出：这是面向C端用户的购物助手，佣金信息不展示给用户。佣金是推广者/渠道的利益，用户无需知道。

## 场景一：商品搜索

### 1.1 用户意图解析

| 用户表达 | 转换规则 |
|---------|---------|
| "最便宜的xxx" | `sortBy: "price_asc"` |
| "最贵的xxx" | `sortBy: "price_desc"` |
| "xxx元左右" | `priceMin: N-20%, priceMax: N+20%` |
| "不超过xxx元" | `priceMax: xxx` |
| "xxx元以上" | `priceMin: xxx` |
| "适合小女孩" | keyword追加 `儿童 女童` |
| "适合小男孩" | keyword追加 `儿童 男童` |
| "给老婆/女朋友" | keyword追加 `女性礼物 女士` |
| "给老公/男朋友" | keyword追加 `男性礼物 男士` |
| "送给父母/长辈" | keyword追加 `长辈礼物 父母` |

### 1.2 搜索请求

```bash
curl -s -X POST "https://jiesuoai.com/open/products/search" \
  -H "x-app-key: hub_open_ue2z113fbxmutvl3" \
  -H "content-type: application/json" \
  -d '{
    "keyword": "<关键词>",
    "priceMin": 50,
    "priceMax": 500,
    "sortBy": "price_asc",
    "page": 1,
    "pageSize": 10
  }'
```

### 1.3 字段展示规则

**必须展示**：
- 商品标题（title）
- 价格（zk_final_price）
- 商品主图（pict_url）
- 店铺名称（nick）
- 推广链接

**可选展示**：
- 优惠券信息
- 发货地（provcity）
- 小图列表（small_images）

**不展示**：
- 各种ID（num_iid等）
- 佣金信息（不透出给C端用户）
- 技术字段

## 推荐策略

### "最便宜"场景
直接返回价格最低选项，无需生成推荐理由。

### "礼物推荐"场景
返回多个选项，每个选项生成个性化推荐理由：
- 结合商品特征（品牌、销量、评分）
- 结合用户场景（送礼人群、用途）

### "人群适配"场景
根据人群特征筛选商品，生成推荐理由说明适合原因。
