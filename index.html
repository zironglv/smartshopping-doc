第一部分：接口增强方案

一、搜索接口 POST /open/products/search

原始接口参数

参数 类型 说明 示例
keyword String 搜索关键词 "蓝牙耳机"
page Integer 页码 1
pageSize Integer 每页数量 10

🆕 新增参数

参数 类型 必填 说明 示例
priceMin Float 否 价格下限 50
priceMax Float 否 价格上限 500
sortBy String 否 排序方式 price_asc / price_desc
sortBy 可选值：
• price_asc - 价格升序（最便宜优先）
• price_desc - 价格降序（最贵优先）
• 默认不传时按综合排序

请求示例

curl -s -X POST "https://jiesuoai.com/open/products/search" \
  -H "x-app-key: hub_open_ue2z113fbxmutvl3" \
  -H "content-type: application/json" \
  -d '{
    "keyword": "蓝牙耳机",
    "priceMin": 50,
    "priceMax": 500,
    "sortBy": "price_asc",
    "page": 1,
    "pageSize": 10
  }'

二、商品详情接口 GET /open/products/{productId}

原始返回字段

字段 说明 示例
id 商品ID cmocqi4og00aw54h2v8ypa43r
title 商品标题 原神联名无线蓝牙耳机...
imageUrl 主图URL https://img.alicdn.com/...
skus[] SKU列表 skuId, skuTitle, specText
offers[] 报价列表 channel, price, shopName

🆕 新增返回字段（变量名与淘宝客原文档一致）

参考文档：taobao.tbk.item.info.get

字段 类型 说明 用户价值 展示规则
num_iid String 商品ID 商品唯一标识 ❌ 内部
title String 商品标题（长标题） 商品完整信息展示 ✅ 展示
short_title String 商品短标题 更简洁友好 ✅ 优先
pict_url String 商品主图 主图展示 ✅ 展示
small_images String[] 商品小图列表 展示更多细节 ✅ 展示
reserve_price Float 商品一口价格 原价展示 ✅ 展示
zk_final_price Float 折扣价 折后价展示 ✅ 优先
user_type Integer 卖家类型（0=集市，1=商城） 判断天猫正品 ✅ 天猫标识
provcity String 商品所在地 发货地信息 ✅ 展示
item_url String 商品链接 商品详情页链接 ❌ 用推广链接
seller_id String 卖家ID 卖家标识 ❌ 内部
volume Integer 30天销量 购买决策参考 ✅ "已售xxx件"
nick String 店铺名称 店铺展示 ✅ 展示
cat_name String 一级类目名称 类目导航 ⚠️ 可选
cat_leaf_name String 叶子类目名称 精细类目 ❌ 不展示
is_prepay Boolean 是否加入消费者保障 消费者保障标识 ⚠️ 可选
shop_dsr Float 店铺DSR评分 店铺质量判断 ✅ "DSR 4.8分"
ratesum Integer 卖家等级 卖家信誉等级 ⚠️ 可选
i_rfd_rate Boolean 退款率低于行业均值 服务质量指标 ⚠️ 可选
h_good_rate Boolean 好评率高于行业均值 服务质量指标 ⚠️ 可选
h_pay_rate30 Boolean 成交转化高于行业均值 销售能力指标 ❌ 不展示
free_shipment Boolean 是否包邮 用户关注成本 ✅ "包邮"
material_lib_type Integer 商品库类型 商品来源标识 ❌ 内部
superior_brand Boolean 是否品牌精选 品牌认证 ✅ "品牌精选"
hot_flag Boolean 是否热门商品 热门标识 ✅ "热门"
sale_price Float 活动价 促销活动价格 ✅ 促销价
kuadian_promotion_info String 跨店满减信息 促销信息展示 ✅ 满减信息
presale_deposit Float 预售定金 预售商品信息 ⚠️ 预售商品
presale_discount_fee_text String 预售优惠信息 预售优惠展示 ⚠️ 预售商品

第二部分：Skill 设计文档

三、触发场景

  • 用户说"找xxx"、"买xxx"、"搜xxx"
  • 用户说"最便宜/最贵的xxx"
  • 用户说"推荐xxx元的礼物"
  • 用户说"适合xxx人群的xxx"

调用配置

HUB_BASE_URL: https://jiesuoai.com
OPEN_API_APP_KEY: hub_open_ue2z113fbxmutvl3

四、用户意图解析规则

用户表达 转换规则
"最便宜的xxx" sortBy: "price_asc"
"最贵的xxx" sortBy: "price_desc"
"xxx元左右" priceMin: N-20%, priceMax: N+20%
"不超过xxx元" priceMax: xxx
"适合小女孩" keyword追加 儿童 女童
"给老婆/女朋友" keyword追加 女性礼物 女士
"给老公/男朋友" keyword追加 男性礼物 男士
"送给父母/长辈" keyword追加 长辈礼物 父母

五、字段展示规则

✅ 必须展示

  • short_title / title - 商品标题（优先短标题）
  • pict_url - Markdown可点击图片
  • zk_final_price / reserve_price - 价格（折后价优先）
  • nick - 店铺名称
  • promotionUrl - 购买链接
  • volume - 30天销量（"已售xxx件"）
  • shop_dsr - 店铺评分（"DSR 4.8分"）
  • free_shipment - 包邮标识
  • user_type - 天猫标识（user_type=1时显示"天猫"）
  • provcity - 发货地

⚠️ 条件展示

  • superior_brand - 为true时显示"品牌精选"
  • hot_flag - 为true时显示"热门"
  • sale_price - 有活动价时显示促销价
  • kuadian_promotion_info - 有跨店满减时显示
  • presale_* - 预售商品时显示预售信息

❌ 不展示（内部字段）

  • num_iid、seller_id、material_lib_type - 内部ID
  • item_url - 用推广链接替代
  • h_pay_rate30 - 技术指标
  • 佣金字段（C端不透出）

七、图片与链接处理规则

图片格式：
必须使用 Markdown 格式：[![标题](pict_url)](推广链接)
• 图片点击跳转到推广链接
• 禁止使用 HTML 标签
链接优先级：
promotionUrl > item_url

八、异常处理

异常 处理方式
搜索无结果 提示"未找到相关商品，建议调整关键词"
API 5xx 错误 重试一次，仍失败则提示"服务暂时不可用"
无推广链接 展示商品信息但提示"暂无购买链接"
