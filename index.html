SmartShopping 接口增强与 Skill 设计

面向 C 端购物 Agent 的 Hub 调用规范，覆盖商品搜索、SKU 筛选、比价推荐等功能。

第一部分：接口增强方案

一、搜索接口 POST /open/products/search

原始接口参数

参数类型说明示例
keywordString搜索关键词"蓝牙耳机"
pageInteger页码1
pageSizeInteger每页数量10

🆕 新增参数

参数类型必填说明示例
priceMinFloat否价格下限50
priceMaxFloat否价格上限500
sortByString否排序方式price_asc / price_desc
sortBy 可选值
  • price_asc — 价格升序（最便宜优先）
  • price_desc — 价格降序（最贵优先）
  • 默认不传时按综合排序

请求示例

curl -s -X POST "https://jiesuoai.com/open/products/search" \
  -H "x-app-key: hub_open_ue2z113fbxmutvl3" \
  -H "content-type: application/json" \
  -d '{
    "keyword": "蓝牙耳机",
    "priceMin": 50,
    "priceMax": 500,
    "sortBy": "price_asc",
    "page": 1,
    "pageSize": 10
  }'

二、商品详情接口 GET /open/products/{productId}

原始返回字段

字段说明示例
id商品IDcmocqi4og00aw54h2v8ypa43r
title商品标题原神联名无线蓝牙耳机...
imageUrl主图URLhttps://img.alicdn.com/...
skus[]SKU列表skuId, skuTitle, specText
offers[]报价列表channel, price, shopName

🆕 新增返回字段（与淘宝客原文档一致）

参考文档：taobao.tbk.item.info.get

字段类型说明展示规则
num_iidString商品ID❌ 内部
titleString商品标题✅ 展示
pict_urlString商品主图✅ 展示
small_imagesString[]商品小图列表✅ 展示
reserve_priceFloat商品一口价格（原价）✅ 展示
zk_final_priceFloat折扣价✅ 优先展示
user_typeInteger卖家类型（0集市/1商城/3特价版）✅ 天猫标识
provcityString商品所在地✅ 发货地
item_urlString商品链接❌ 用推广链接
seller_idString卖家ID❌ 内部
volumeInteger30天销量✅ "已售xxx件"
nickString店铺名称✅ 展示
cat_nameString一级类目名称⚠️ 可选
cat_leaf_nameString叶子类目名称❌ 不展示
is_prepayBoolean是否加入消费者保障⚠️ 可选
shop_dsrFloat店铺DSR评分✅ "DSR 4.8分"
ratesumInteger卖家等级⚠️ 可选
i_rfd_rateBoolean退款率低于行业均值⚠️ 可选
h_good_rateBoolean好评率高于行业均值⚠️ 可选
h_pay_rate30Boolean成交转化高于行业均值❌ 不展示
free_shipmentBoolean是否包邮✅ "包邮"
material_lib_typeInteger商品库类型❌ 内部
superior_brandInteger是否品牌精选（0否/1是）✅ "品牌精选"
hot_flagInteger是否热门商品（0否/1是）✅ "热门"
sale_priceFloat活动价✅ 促销价
kuadian_promotion_infoString跨店满减信息✅ 满减信息
presale_depositFloat预售定金⚠️ 预售商品
presale_discount_fee_textString预售商品优惠信息⚠️ 预售商品
⚠️ 字段来源说明

以上字段名均与淘宝客官方 API taobao.tbk.item.info.get 响应参数完全一致，未做任何改动。

第二部分：Skill 设计

三、触发场景

  • 用户说"找xxx"、"买xxx"、"搜xxx"
  • 用户说"最便宜/最贵的xxx"
  • 用户说"推荐xxx元的礼物"
  • 用户说"适合xxx人群的xxx"

调用配置

HUB_BASE_URL: https://jiesuoai.com
OPEN_API_APP_KEY: hub_open_ue2z113fbxmutvl3

四、用户意图解析规则

用户表达转换规则
"最便宜的xxx"sortBy: "price_asc"
"最贵的xxx"sortBy: "price_desc"
"xxx元左右"priceMin: N-20%, priceMax: N+20%
"不超过xxx元"priceMax: xxx
"适合小女孩"keyword追加 儿童 女童
"给老婆/女朋友"keyword追加 女性礼物 女士
"给老公/男朋友"keyword追加 男性礼物 男士
"送给父母/长辈"keyword追加 长辈礼物 父母

五、字段展示规则

✅ 必须展示
  • title — 商品标题
  • pict_url — Markdown 可点击图片
  • zk_final_price / reserve_price — 价格（折后价优先）
  • nick — 店铺名称
  • promotionUrl — 购买链接
  • volume — 30天销量（"已售xxx件"）
  • shop_dsr — 店铺评分（"DSR 4.8分"）
  • free_shipment — 包邮标识
  • user_type — 天猫标识（=1时显示"天猫"）
  • provcity — 发货地
⚠️ 条件展示
  • superior_brand — =1 时显示"品牌精选"
  • hot_flag — =1 时显示"热门"
  • sale_price — 有活动价时显示促销价
  • kuadian_promotion_info — 有跨店满减时显示
  • presale_* — 预售商品时显示预售信息
❌ 不展示（内部字段）
  • num_iid、seller_id、material_lib_type — 内部ID
  • item_url — 用推广链接替代
  • h_pay_rate30 — 技术指标
  • 佣金字段 — C端不透出

图片与链接规则

图片格式

必须使用 Markdown 格式：[![标题](pict_url)](推广链接)

  • 图片点击跳转到推广链接
  • 禁止使用 HTML 标签
链接优先级

promotionUrl > item_url

七、异常处理

异常处理方式
搜索无结果提示"未找到相关商品，建议调整关键词"
API 5xx 错误重试一次，仍失败则提示"服务暂时不可用"
无推广链接展示商品信息但提示"暂无购买链接"
